3.疑似クラスおよび疑似要素によるセレクタ

   

疑似クラスによるセレクタ

    

・a:link（未訪問のリンク要素の指定）


a:link{
    color:pink;
}
aタグに対して、そのリンクが未訪問の状態である場合を指定してスタイルを適用します。
    


・a:visited（訪問済のリンク要素の指定）


a:visited{
    color:pink;
}
aタグに対して、そのリンクが訪問済の状態である場合を指定してスタイルを適用します。
    
     

・:hover（要素にマウスがホバーした際の指定）


a:hover{
    color:pink;
}
要素に対してマウスがホバーした状態を指定してスタイルを適用させます。
    
    
    

・a:active（要素がアクティブ状態になった際の指定）


a:active{
    color:pink;
}
要素がアクティブの状態の場合を指定してスタイルを適用させます。
    
aタグではマウスを左クリックしてから離すまでがアクティブ状態となります。

上記四つのセレクタの表示例

    

・親要素　要素:nth-child(n)（親要素内のn番目の要素の指定）


親要素　要素:nth-child(n){
    color:pink;
}
親要素内にあるすべての要素の中でn番目にある要素にスタイルを適用させます。

表示例

  


・親要素　要素:nth-of-type(n)
    （親要素内の同じ種類の要素の中のn番目の要素）


親要素　要素:nth-of-type(n){
    color:pink;
}
親要素内にある同じ種類の要素の中でn番目にある要素に
スタイルを適用させます。

表示例

    
    

・親要素:not(要素1)（要素1以外の要素の指定）


親要素:not(要素1){
    color:pink;
}
親要素内の指定の要素1を除外してスタイルを適用させます。

表示例

 

    
疑似要素によるセレクタ

    

・属性値:: first-letter（要素の一文字目を指定）


p.属性値::first-letter{
    color:pink;
}
pタグ要素に属性値が反映され、pタグで囲ったテキストの一文字目に適用されます。

表示例




・属性値::first-line（要素の一行目を指定)


p.属性値::first-line{
    color:pink;
}
pタグ要素に属性値が反映され、pタグで囲ったテキストの一行目に適用されます。

表示例

    
    

・属性値::before（要素の直前にコンテンツを追加）


p.属性値::before{
    content:"要素の直前にコンテンツを追加";
    color:pink;
}
first-letterやfirst-lineと違う点は、contentプロパティが必要ということです。
    
contentの中身にテキストを指定する場合は、ダブルコーテーションもしくはシングルコーテーションで囲みます。
またfont-sizeやcolorなどのプロパティ設定は、追加するテキストに対して適用されます。

表示例

    
    

・属性値::after（要素の直後にコンテンツを追加）


p.属性値::after{
    content:"要素の直後にコンテンツを追加";
    color:pink;
}
before、afterともにcontentプロパティに画像を指定すると、
テキストだけでなく画像も追加することが可能です。
contentの中身が画像の場合はダブルクォーテーションで
囲む必要はありません。

    

4.タイプセレクタ(div p header,, ete) 

タイプセレクタとはhead、body、h1、p など既存の要素に対してスタイルを設定し、設定された要素内（親要素）に
あるすべての要素（子要素）にその設定されたスタイルが適用されるセレクタをタイプセレクタと呼びます。
