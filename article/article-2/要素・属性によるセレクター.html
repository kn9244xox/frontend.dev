1.要素によるセレクター

指定したい要素が1つの時


h1{
      Color:pink;
}
指定したい要素名を１つだけ、記して色を変える。

表示例



指定したい要素が2つ以上の時


h1, P{
        Color:pink;
}
要素と要素、h1とpの間に, (カンマ)を入れてセレクターを指定すると、h1とpの要素にデザインが適応されます。

表示例



指定したい要素が要素の中にある時


h1 p{
      Color:pink;
}
要素と要素の間に半角スペースを入れてセレクタを、指定するとh1の中のpを指定できます。

表示例



親要素内の一階層下の子要素を指定したい時


h1>p{
      Color:pink;
}
要素と要素の間に > を入れてセレクタを指定すると、h1に囲まれている一番最初の要素pを指定できます。

表示例



指定したい要素の後ろにある同じ階層の要素を指定したい時


h1~p{
      Color:pink;
}
要素と要素の間に~を入れてセレクタを指定すると、要素h1の後ろにある同じ階層の要素pを全部指定できます。

2.属性によるセレクター





・要素名[属性名]


p［class］{
    color:pink;
}
class属性があるpタグのみ指定できます。

表示例

     

・要素名[属性名=”属性値”]

        

p［class="pink"］{
    color:
}
指定の属性が指定の属性値を持っている要素のみ指定できます。

表示例

    

・要素名[属性名~=”属性値”]


p［class~="pink"］{
    color:
}
=（イコール）の前に~（チルダ）を記述すると、指定の属性が指定の属性値を含んでいる要素のみ指定できます。

表示例

    

・要素名[属性名^=”文字列”]
 

p［class^="red"］{
    color:
}
=（イコール）の前に^（キャレット）を記述すると、指定の文字列から始まる属性値を持っている要素を指定できます。

表示例

    

・要素名[属性名$=”文字列”]
 

p［class$="001"］{
    color:
}
=（イコール）の前に$（ドル）を記述すると、指定の文字列で終わる属性値を持っている要素を指定できます。
 
表示例



・要素名[属性名*=”文字列”]


p［class*="001"］{
    color:
}
=（イコール）の前に*（アスタリスク）を記述すると、指定の文字列を含む属性値を持っている要素を指定できます。

表示例
